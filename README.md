# RedstoneLimiter

Datapack для ограничения redstone-механизмов и снижения lag на ванильных серверах.

## Установка

1. Запусти Minecraft.
2. Выбери нужный мир → **Настроить**.
3. Нажми **Открыть папку мира**.
4. Перейди в папку `datapacks`.
5. Положи архив **RedstoneLimiter** в эту папку.
6. Зайди в мир.
7. Выполни команду:
   /function antishlag:load
8. Готово.

## Проверка установки

Выполни команду:
   /datapack list

Datapack **RedstoneLimiter** должен быть в статусе `enabled`.

## Требования

- Minecraft Java Edition  
- Версия: см. описание репозитория
